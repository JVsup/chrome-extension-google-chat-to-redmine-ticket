Privacy Policy for "Chat to Redmine Ticket"

Effective Date: March 28, 2026

1. Data Collection and Usage

The "Chat to Redmine Ticket" extension is designed to read user-selected text from Google Chat and convert it into a structured Redmine ticket using AI. The extension temporarily accesses personal communications and website content solely for the purpose of executing the user's explicit command (creating a ticket from highlighted text).

2. Data Storage

All user configurations, including Redmine server URLs, API keys (for Redmine, OpenAI, Anthropic, Google AI Studio, Vertex AI, or custom local LLMs), and default project settings, are stored locally on the user's device using Chrome's secure local storage API.

The developer of this extension does not collect, transmit, or store any of this configuration data, nor any chat content, on their own servers.

3. Third-Party Services

To function as intended, the extension transmits the user-selected text and the associated API keys directly to the user-configured third-party AI providers and the user's configured Redmine server. This communication happens directly between the user's browser and the respective APIs. The extension does not intercept, log, or reroute this data.

4. Data Sharing

We do not sell, trade, or otherwise transfer any user data to outside parties. Data is only transmitted to the specific APIs explicitly configured and authorized by the user within the extension's settings.

5. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated effective date.
